# rideau-canal-sensor-simulation
### Azure IoT Hub Configuration

Three IoT devices were registered in Azure IoT Hub to represent sensor locations along the Rideau Canal.

- dows-lake
- fifth-avenue
- nac

Each device uses symmetric key authentication and sends telemetry data every 10 seconds.
